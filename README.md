# Pharmacy Management System

A modern **Pharmacy Management System** built with **Next.js**, **TypeScript**, and **MongoDB** to manage medicines, inventory, sales, and pharmacy operations efficiently.

---

## 🚀 Features

* Medicine inventory management
* Sales and billing system
* Customer and prescription records
* Stock tracking and alerts
* Secure authentication system
* Admin dashboard
* Responsive UI design

---

## 🛠️ Tech Stack

* **Framework:** Next.js
* **Language:** TypeScript
* **Database:** MongoDB
* **Styling:** Tailwind CSS (or your choice)
* **Authentication:** NextAuth / JWT (if used)
* **API:** Next.js API Routes

---

## 📦 Installation

```bash
git clone https://github.com/your-username/pharmacy-management-system.git
cd pharmacy-management-system
npm install
```

---

## ▶️ Run Project

```bash
npm run dev
```

Open: `http://localhost:3000`

---

## ⚙️ Environment Variables

Create a `.env.local` file:

```
MONGODB_URI=your_mongodb_connection_string
NEXTAUTH_SECRET=your_secret_key
NEXTAUTH_URL=http://localhost:3000
```
